# Privacy Policy

**Effective Date: March 1, 2026**

Kingfisher respects your privacy. This policy explains what data we collect and why.

## 1. Data Collection

Kingfisher is designed to work entirely offline. Kingfisher currently sends no telemetry.

### Telemetry
Kingfisher currently sends no automatic telemetry, analytics, crash uploads, or installation pings. All analysis stays local on your device.

## 2. What We NEVER Collect
- **NO Filenames or Folder Paths.**
- **NO Image Content or Thumbnails.**
- **NO Personally Identifiable Information (PII)** like names, emails (unless provided in a feedback form), or location data.

## 3. Data Usage
Because telemetry is disabled, there is no background usage data to sell or share.

## 4. Contact
If you have questions about this policy, open a GitHub issue in the Kingfisher repository.
